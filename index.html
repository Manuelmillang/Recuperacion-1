Aplicacions Web - Examen REC Bootstrap i JS - 07/03/2024

  1. Classes Bootstrap (15 punts)

    Heu de posar l'enllaç i les classes Bootstrap per:

    • Header: centrat, amb marge top, texte gran
    • Main: padding i margin, border blau, ben redondejat
    • Botons: Classes de botons i amb colors diferenciats

  2. Taules de multiplicar (25 punts)

    Donat un número sencer, desenvolupar la seva taula de multiplicar (del 0 fins al 10)

    • Els resultats s'han de mostrar dins un table HTML.
    • Heu de mostrar 3 columnes: una pel número que ha introduit l'usuari (multiplicant), un altre pel multiplicador (del 0 al 10), i una tercera pel resultat.
    • Heu d'avisar a l'usuari si no ha escrit cap valor a l'input.
    • El botó "Neteja" esborra la taula

  3. Generació de colors CSS aleatoris (40 punts)

    • Heu de fer una funció de generació ràpida de colors rgb aleatoris i mostrar-los a la capsa
    • La generació comença quan es pitja "Start" i s'atura quan es pitja "Stop"
    • El botó Start és el mateix que el de Stop. Heu de controlar que quan es pitja, ha de canviar el seu texte i el seu color.
    • Quan la generació s'atura a un color, aquest s'ha de posar com a fons del main#principal
    • El botó reinicia atura la generació i lleva el color de fons.


  4. Emplenar un Array, ordenat i sense repeticions (20 punts)

    L'usuari afegeix elements numèrics a un array i aquest es mostra per pantalla ordenat i sense repeticions

    • Heu d'afegir els elements escrits a l'input de tipus numèric.
    • Cada vegada que s'afegeix un número, si no està repetit, heu d'actualitzar l'array, ordenar-lo i mostrar-lo al resultat
    • Si no s'ha escrit cap número o si el número està repetit, no s'ha de fer res, ni tan sols avisar l'usuari
    • El botó "Neteja" esborra el contingut del Array.
